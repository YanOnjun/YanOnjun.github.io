继承

发表于 2023-05-29 阅读次数：

不同类型的对象，相互之间有一些共同点。同时每一个对象又定义了额外的特性使他们与众不同。继承是使用父类的定义作为基础建立子类的思想，子类拥有父类的特征和行为，子类可以增加新的数据或者新的功能，但是不能选择性的继承父类，只能全部继承。

继承符合的关系是 is-a

使用继承可以快速创建新的类，提高代码重用性，程序的可维护性。

特性：

  1. 子类拥有父类所有的属性和方法，但是不能访问私有属性和方法。
  2. 子类可以拥有自己的属性和方法。
  3. 子类可以重新实现父类的方法。
  4. 所有类默认继承 Object（Java)
  5. Java 的继承是单继承，但是可以多重继承

多重继承

多重继承指的是一个父类还有一个父类，子类也可以继承到父类的父类的属性和方法（C++中没有这个特性）

实现

实现是继承的一种，是类与[[接口]]之间的继承关系，由于接口是一组方法的集合，所以用实现方法而不是继承方法更符合描述。

构造器

Java: 子类不继承父类的构造器，只是在实例化时显式或隐式调用。父类如果有有参构造器，那么子类的构造器中必须显式的使用 super 方式调用，如果是无参构造器（Java默认没有写构造器的话也有一个无参构造器），则不需要调用，系统会自动调用父类的无参构造器。
