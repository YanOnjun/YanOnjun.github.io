Java 特点

java语言特性

  • 面对对象：将重点放在数据和对象的接口上
  • 简单性：相比c++，减少了头文件、指针等很多概念，而且java可以在很小的机器上独立运行
  • Java语言是健壮的 : 强类型机制, 异常处理, 垃圾的自动收集
  • 跨平台性：java文件被编译成字节码文件，然后在不同的机器上可以通过JVM编译成机器码执行
  • 解释性语言：需要解释器来执行
  • 分布式：有丰富的例程库用于处理TCP/IP协议，可以通过URL访问网络上的对象
